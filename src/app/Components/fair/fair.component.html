Fair System

THIS MECHANISM WORKS FOR BOTH COIN FLIP AND ROULETTE!

To keep our rounds fair and truly random, we utilize the provably fair system.

At the beginning of a round, the server generates a salt, and a winning ticket to identify the round. These two elements are concatenated to create a round hash as follows:

Example
Salt : bDJpd0VVRC9hb3dqdG1sdGhKbFBtdz09
Winning Ticket (Random) : 22.51806220891106
Hash : sha256([Salt]:[Ticket])

When the round is complete, the server will choose a winner using the winning ticket which are choosed randomly.

After choosing a winner, the server will reveal the round's salt and winning ticket allowing you to back calculate the other quantities as necessary.

To ensure this system is working properly, we have implemented a client side calculator for you to verify any completed round.
